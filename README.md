<h1>Смена порядка букв в каждом слове на обратный</h1>

<p>Описание: Приложение с минимальным интерфейсом, состоящим из одного input поля, в которое можно вписать строку.
 В строке после срабатывания скрипта меняется порядок букв на обратный с некоторыми условиями:
  <ul>
    <li>Cохранение регитсра, например: Мышь -> Ьшым, houSe -> esuOh</li>
    <li>Сохранение пунктуации, например: is 'cold' now -> si 'dloc' won </li>
    <li>Составные слова считаются отдельными словами, например: third-part -> driht-trap</li>
  </ul>
</p>

<p>Какое-то время будет доступ к просмотру функционала по <a href="http://31.28.27.170:8000/index.php">ссылке</a></p>

<h2>Инструкция по разворачиванию</h2>

<ol>
  <li>Клонировать репозиторий</li>
  <li>В локальном репозитории выполнить следующие комманды:
    <ol>
      <li>Запустить сборку образов <br> <code>docker compose build</code></li>
      <li>Установить composer зависимости <br> <code>docker compose run --rm composer install</code></li>
      <li>Запустить сервер <br> <code>docker compose up php-app</code></li>
      <li>Зайти на <strong>http://localhost:8000<strong> для просмотра самой программы</li>
      <li>Запустить тесты <br> <code>docker compose run --rm php-tests</code></li>
    </ol>
  </li>
</ol>
